1. 담당자가 업무 요청을 한다. (담당자 -> 개발자)

3. 업무 요청을 확인하고, (본인이) 이해한 내용을 정리한다.
 
4. 정리한 내용을 미팅 혹은 메신저, 메일 등으로 담당자에게 설명한다. (개발자 -> 담당자)
 
5. 작업 일자를 확인한다. (버퍼 타임을 꼭 넣도록 한다. 조정 필요시 요청한다.)
 
6. 최종적으로 정해진 내용을 메일 혹은 문서로 만들어 담당자에게 다시 한번 전달하고 확인받는다.
